Návod ke spuštění obou programů:

Instalace Pythonu:
- pokud máte nainstalovaný Python ve svém počítači, tak můžete část Instalace Pythonu přeskočit
    1. Je nutné stáhnout a nainstalovat Python (nejnovejsi verzi 3.11) z odkazu - https://www.python.org/downloads/
    2. Během instalace zaškrtnout možnost - Add Python to PATH

Instalace knihoven:
- je nutné doinstalovat balíček Pandas https://pypi.org/project/pandas/
    1. Nainstalovat Pandas balíček
        - Windows/Linux přes konzoli/terminál: pip install pandas
        - existuje druhá možnost, kde vám IDE jako PyCharm samo nabídne instalaci přes nápovědu

Spuštění programů:
- jednotlivé programy jsou ve dvou adresářích - Winston a Maze
    1. Otevřít si v konzoli adresář se zdrojovými kody vybraného programu
    2. spustit program: 
        - příkaz do konzole/terminálu: python main.py nebo python3 main.py
    3. Spustí se program


Poznámka:
- programy je také možné spustit přes IDE jako PyCharm, stačí pomocí IDE otevřít zdrojový kod, zvolit intepret
  pythonu pro daný soubor a spustit přes Play
